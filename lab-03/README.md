# Лабораторна робота №3: Розумне кешування (Мемоїзація)

## Де що лежить
- `lib/memoize.js` — сама функція, яка обгортає інші функції та дає їм пам'ять.
- `lib/strategies.js` — тут живуть алгоритми очищення (LRU, LFU, TTL).
- `example/example.js` — файл з тестами, щоб подивитися, як усе це працює на практиці.

## Для запуску ввести в термінал:
node example/example.js
