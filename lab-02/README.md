## Структура файлів
* `generators.js` — тут лежать самі генератори (числа та кольори).
* `consumer.js` — тут лежить функція з таймером.
* `index.js` — головний файл, який збирає ці модулі разом.
* Папка `example/` і файл `example.js` — це окремий міні-проєкт. Він підключає мою бібліотеку і просто показує, як вона працює.

Щоб перевірити роботу, відкрийте термінал у папці `lab-02` і введіть команду:

node example/example.js
