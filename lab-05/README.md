# Лабораторна робота №5: Асинхронний Map

## Опис проєкту
У цій лабораторній роботі реалізовано власні асинхронні варіанти стандартної функції масиву `map`. На відміну від базового `map`, ці функції здатні коректно обробляти асинхронні операції (таймери, мережеві запити тощо) для кожного елемента масиву.

## Реалізовані підходи
1. **Callback-based версія (`asyncMapCallback`):** Виконує паралельну обробку елементів та повертає підсумковий масив через фінальний callback-обробник.
2. **Promise-based версія (`asyncMapPromise`):** Сучасна реалізація, що використовує `Promise.all` та підтримує синтаксис `async/await`.

## Структура файлів
- `lib/asyncMap.js` — реалізація логіки функцій.
- `example/example.js` — демо-кейси для перевірки всіх режимів роботи.

## Запуск тестів
Для перевірки роботи функцій виконайте команду в терміналі:

node example/example.js
