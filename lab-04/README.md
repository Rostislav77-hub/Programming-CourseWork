# Лабораторна робота №4: Bi-Directional Priority Queue

## Основні можливості
Клас `PriorityQueue` підтримує методи `peek` (перегляд) та `dequeue` (вилучення), які приймають об'єкт з прапорцями:
- `highest: true` — повертає елемент з найвищим пріоритетом (найменше число).
- `lowest: true` — повертає елемент з найнижчим пріоритетом (найбільше число).
- `oldest: true` — повертає найстаріший елемент (поведінка звичайної черги FIFO).
- `newest: true` — повертає найновіший елемент (поведінка стеку LIFO).

## Структура
- `lib/PriorityQueue.js` — реалізація логіки двосторонньої черги.
- `example/example.js` — тестові сценарії для перевірки всіх 4-х критеріїв.
